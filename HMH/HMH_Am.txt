## Test reports

### On local

To view the report on local you can type in `yarn workspace @hmhco/lift-e2e testReport` when in `lift` root.

We use `allure-reports` for E2E report generation, after the test run, the `teardown.js` script uses `allure-commandline` to generate the report in html and stores it in the `allure-reports` folder.
The generated report cannot be opened directly from file, as modern browsers have CORS limitations that get set off by the nature of the scripts in the report html.
